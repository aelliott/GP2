Detailed Description

The PrettyTabMenu class represents each individual menu making up the whole of the PrettyTabBar.

The structure of the PrettyTabMenu is an optional title followed by a sequence of tabs. A title is expected to be provided for every menu apart from the first menu constructed internally (the "default" menu) - and this is reflected in that any tab menu constructed without a title will return "default" in title() and will send "default" in the tabTriggered() signal.

To add a tab to the menu you would call addTab() and make note of the integer returned, this is the integer which will be reported by the tabTriggered() signal along with the title of the menu.

Since another tab menu can take focus from one of the tabs in this menu the menu provides a clearSelection() method which should be called whenever a tab in another menu takes focus to ensure that only one tab is "active" at once.
