bool Developer::Program::open ( ) [virtual]

Open the file into the protected variable _fp.

This sets up everything that is generic to all files, it is intended that this method should be overridden in derived classes, which will initial call this function via GPFile::open() and then perform logic specific to the type of file opened.

Returns:
Boolean, true if opened successfully, false otherwise

bool Developer::Program::save ( ) [virtual]

Save this file to the current active file.
