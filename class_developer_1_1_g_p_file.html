Detailed Description

Abstract base class for GP project files.

The GPFile class provides a generalised interface for querying and changing the state of files in a GP project. The class maintains a FileStatus member variable which informs the rest of the system whether the file needs to be saved, reloaded, etc.

In addition it provides an interface for those tasks. The save() function saves any pending changes to the file contained in _path (or prompts the user for a location if one is not set), and saveAs() saves the file to a new location given by its argument.

void Developer::GPFile::statusChanged ( FileStatus  status) [signal]

This signal is emitted whenever it is detected that the file has changed in any way.

This change can be triggered by normal editing within the IDE, edits made outside of the running editor or the file being deleted from the filesystem.

In cases where more than one applies the order of precedence should be

  1. Error
  2. Deleted
  3. ExternallyModified
  4. Modified
  5. Normal
Parameters:
statusThe new status of the file
