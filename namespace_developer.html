enum Developer::GPVersions

The GPVersions enum specifies the types of GP program allowed.

This enumeration exists to allow for extension of the editor to new as yet unspecified versions of GP by allowing them to be defined as a new enumeration value. This means that if for example the language were experimentally altered to include colouring semantics then a value such as ColouredGP2 could be inserted to handle this.

Enumerator:
GP1 

Greg Manning's GP.

GP2 

GP2, without rooted graph programs.

RootedGP2 

GP2, with rooted graph programs.
